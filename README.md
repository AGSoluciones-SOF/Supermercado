# 📌 Descripción

Supermercado en Línea es una plataforma web diseñada para la venta de productos de supermercado mediante Internet.

El sistema permite administrar productos, categorías, clientes, pedidos y ventas desde un panel administrativo moderno y responsive.

Está pensado como una solución para supermercados, tiendas de conveniencia, minisúpers y negocios de venta al público.
